# Netflix Interactive Dashboard

## Objective
Build an interactive dashboard using Streamlit to analyze Netflix Movies and TV Shows dataset.

---

## Tools & Technologies Used
- Python
- Streamlit
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Dashboard Features
- Upload CSV dataset
- Dataset preview
- Missing value analysis
- Movies vs TV Shows visualization
- Release year trend analysis
- Rating distribution
- Top countries analysis
- Genre analysis

---

## 📈 Key Insights
- Netflix contains more Movies than TV Shows.
- Content increased rapidly after 2015.
- The United States contributes the highest amount of content.
- TV-MA is one of the most common ratings.
- Drama and International Movies are among the top genres.

---

## How to Run the Project

Install required libraries:

pip install streamlit pandas matplotlib seaborn

Run the dashboard:

python -m streamlit run app.py

---

## Conclusion
This dashboard helps analyze Netflix content trends interactively using Streamlit visualizations.
